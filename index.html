소개

보기 좋은 화면보다 한 걸음 더, 계속 쓰기 좋은 화면을 만듭니다.

저는 화면의 첫인상과 실제 사용 흐름을 함께 봅니다. 사용자가 다음 행동을 자연스럽게 찾을 수 있도록 정보 구조를 정리하고, 작은 인터랙션까지 일관되게 다듬습니다.

구현에서는 시맨틱 마크업, 예측 가능한 스타일 규칙, 명확한 컴포넌트 경계를 중요하게 둡니다. 빠르게 만들되 나중에 고치기 쉬운 상태로 남기는 것을 좋은 결과라고 생각합니다.

기술

제품의 속도와 품질을 같이 높이는 실용적인 역량.

01

프론트엔드 구현

시맨틱 HTML, 현대적인 CSS, JavaScript, React 기반 화면 구현과 브라우저 디버깅.

02

사용자 경험

폼, 대시보드, 목록, 상세 화면의 흐름을 정리하고 상태와 피드백을 명확하게 설계.

03

품질과 운영

반응형 검수, 접근성 점검, 성능 최적화, 문서화와 GitHub Pages 배포 경험.
